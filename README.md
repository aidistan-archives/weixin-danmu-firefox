# firefox-weixin-danmu

微信网页弹幕Firefox插件

## 安装

### 从 Firefox 附件商店安装 (较易)

Firefox 附件商店链接：[https://addons.mozilla.org/zh-CN/firefox/addon/weixin-danmu/](https://addons.mozilla.org/zh-CN/firefox/addon/weixin-danmu/)

制作中...

## 使用

1. 安装插件
2. 在火狐浏览器的任意一个标签中登陆网页版微信
3. 打开要捕获消息的群或好友的对话框
4. 切换到要显示弹幕的任意一个网页
5. 在手机微信上向所选的好友或群发消息
6. 弹幕出现！
